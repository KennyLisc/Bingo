# flutter_bingo

Bingo 游戏flutter实现，flutter redux全局状态管理学习项目，
主要功能：Bingo游戏实现（可以排序查看），主题切换，语言切换，小方框广告显示以及跳转

所用插件：

flutter_redux: ^0.5.2

shared_preferences: ^0.4.2

event_bus:  ^1.0.1

animated_text_kit: ^1.1.0

url_launcher: ^3.0.1
